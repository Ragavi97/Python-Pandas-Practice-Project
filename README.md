# Python-Pandas-Practice-Project

This repository contains a practice project for exploring and learning Python Pandas. It focuses on basic data manipulation and exploration techniques using DataFrames, iloc, loc, unique, group by, merge, pivot and other fundamental Pandas functions.

<h2>Features</h2>
<ul>
  <li>Load datasets using <code>pandas.read_csv()</code></li>
  <li>Explore datasets using <code>.head()</code>, <code>.tail()</code>, <code>.info()</code>, <code>.describe()</code></li>
  <li>Select data using <code>iloc</code> (index-based) and <code>loc</code> (label-based)</li>
  <li>Basic filtering and slicing of data</li>
  <li>Simple aggregation and summary functions</li>
</ul>
